\section{Mehrspieler-Features}
Konzepte, die es ermöglichen mit mehreren Spielern gleichzeitig zu spielen, bzw. Interaktionen zwischen ihnen schaffen.

\subsection*{Synchronisation zwischen mobilen Endgeräten}\label{sec:synchronisation-zwischen-mobilen-endgeraeten}

Damit alle Spieler miteinander spielen können, ist es wichtig, dass sie miteinander, entweder direkt oder über einen Server, kommunizieren können. Zudem ist es für die meisten Spiele erforderlich, dass die einzelnen sich im Spiel befindlichen mobilen Geräte synchronisiert werden. Das heißt, dass mindestens eine Instanz die Position aller teilnehmenden Geräte und ihren zu teilenden Informationen kennt. Es ist wichtig, dass Daten, die jeder sehen kann, bei jedem identisch und zur selben Zeit angezeigt werden.
Um dies verwirklichen zu können ist z.B. eine Server-Client-Kommunikation (siehe Abschnitt \ref{kommunikation}) erforderlich.


\subsection*{Chat}\label{sec:chat}
Um mit seinen Teammitgliedern oder dem gegnerischen Team zu kommunizieren gibt es eine Reihe von Möglichkeiten. Wenn man sich außer Sicht- und Hörweite befindet, kann dies durch mobile Geräte stattfinden. Zum Beispiel indem Nachrichten versendet werden, die für alle Chatgruppenteilnehmer (beispielsweise für das eigene Team, oder für alle Mitspieler) zu sehen sind.
Auch hierfür ist eine Server-Client-Kommunikation (siehe Abschnitt \ref{kommunikation}) und zur Darstellung die GUI (siehe Abschnitt \ref{gui}) erforderlich.
