Was Github bereitstellt

  1. git
  2. github-pages
  3. wiki

git

  1. "Rückrat" von Git
  2. Entwickelt für Linux-Kernel
  3. fork
    • komplette Kopie des Repository
    • selbst weiterentwickeln und verbreiten
    • Änderungen/Erweiterungen zurückfließen lassen
  4. branch
    • ermöglicht verschiedene Versionen
    • Webpage hosten (gh-pages)
    • Aufnahme in master-branch möglich
  5. pull request
    • Anfrage um Änderung/Erweiterung aufzunehmen
    • kommentierung möglich
    • wird von der zuständigen Person genehmigt oder abgelehnt

github-pages

  • hosten von "statischen" Webseiten
  • Präsentation für Projekt

wiki

  • Arbeitskoordinierung
  • Festhalten des Projektstands
  • Dokumentationsmöglichkeit während Entwicklung
